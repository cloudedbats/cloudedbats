Species

Scientific names and species lists are important for biological monitoring. IUCN maintains a global red list of threatened species, and for some species groups the list is comprehensively assessed. Mammals are in such a group and it can therefore be used for most of the bats. As always, there are different opinions about scientific names and species classification, but this list may work as a first try as a taxonomic backbone for CloudedBats.

Filters: It is possible to apply filters to the shown list of species. Filters are available for countries, families, genus and red list categories.

Species table: The species table is sorted on scientific names. The rows in the family, genus and scientific name columns are linked by name to Wikipedia. The 'IUCN Red List' field are linked by id to IUCN, and the rows in the EOL column are linked by name to Encyclopedia of Life.

Download: The species table can be downloaded as a tab separated text file. This type of files can be opened in a spread sheet program like Microsoft Excel or LibreOffice Calc.

Final note: When reaching this point, your data can be shared globally. To learn more, search for keywords like DarwinCore, DarwinCore-Archive (DwC-A), IPT, GBIF, TDWG or Altas of Living Australia (ALA). GBIF and ALA are also free and open software. Clones of ALA are set up by natural history museums in other countries. Many of the national museums already have their own IPT servers to share data in the DwC-A format.
